!! Installation Guide !!

--> Just follow these steps, if you have no idea how to run the project.

Step-01 Make sure extracted file should be in one folder (if there are subfolder put the files in one folder).

Step-02 Put the extracted folder (BBMS) in xampp/htdocs folder.

Step-03 Now your project files should be here - xampp/htdocs/BBMS/project_files.

Step-04 Now Open your xampp serve and start mysql and apache services.

Step-05 Make sure apache and mysql services are running properly.

Step-06 Now go to your browser and type: localhost

Step-07 Go to phpmyadmin

Step-08 Create a database named 'bbms'

Step-09 Now go to created database and click on 'import' tab.

Step-10 Select database file (.sql file) that i have provided (Check database file folder)

Step-11 Click on import button on the bottom.

Step-12 Now go to your browser and type: localhost/bbms/index.php

Step-13 That's all.

Step-15 Now you can login or register as an donor or patient.
